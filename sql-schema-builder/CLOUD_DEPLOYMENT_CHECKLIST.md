# Чек-лист: Заливка веб-приложения на облако

## Подготовка приложения

### 1. Сборка приложения
```bash
# Установка зависимостей
npm install

# Сборка для продакшена
npm run build
```

После сборки появится папка `dist` с готовыми файлами.

### 2. Проверка
- Приложение запускается без ошибок
- Все функ работает в продакшен-режиме

---

## Варианты развертывания

### Рекомендуемые бесплатные варианты:

#### 1. Vercel (лучший для Node.js)
**Плюсы:** Бесплатно, автоматический CI/CD, идеально для Node.js
**Минусы:** Ограничение на 100 часов в месяц (для бесплатного плана)

**Как залить:**
```bash
# Установка Vercel CLI
npm i -g vercel

# Деплой
vercel
```

Или через GitHub: подключить репозиторий в Vercel.

---

#### 2. Render
**Плюсы:** Бесплатно для Node.js, автоматический деплой
**Минусы:** "Спит" после 15 минут бездействия

**Как залить:**
1. Зарегистрируйтесь на render.com
2. Подключите GitHub репозиторий
3. Выберите "Web Service"
4. Настройте:
   - Build Command: `npm run build`
   - Start Command: `npm run dev` или настройте продакшен сервер

---

#### 3. Railway
**Плюсы:** Простой интерфейс, много возможностей
**Минусы:** Платно после бесплатного лимита ($5/месяц)

**Как залить:**
1. Зарегистрируйтесь на railway.app
2. Подключите GitHub
3. Выберите проект для деплоя

---

#### 4. Netlify (альтернатива)
**Плюсы:** Бесплатно, отличный интерфейс
**Минусы:** Больше подходит для статических сайтов

---

## Варианты для учебного заведения

### Если есть сервер с поддержкой Node.js:
1. Собрать приложение: `npm run build`
2. Скопировать папку `dist` на сервер
3. Настроить Nginx как прокси

### Если используется PM2:
```bash
# Установка PM2
npm install -g pm2

# Запуск
pm2 start npm --name "sql-schema-builder" -- run dev

# Сохранение конфигурации
pm2 save
```

## Частые проблемы

### 1. Приложение не запускается
**Решение:** Проверьте порт и Node.js версию

### 2. Статические файлы не грузятся
**Решение:** Проверьте base URL в конфигурации

### 3. База данных не подключается
**Решение:** Настройте переменные окружения
